# 🚗 Smart Indoor Parking System

A smart parking management system that digitizes the process of managing indoor parking lots. Instead of manually assigning parking spaces, drivers can scan a QR code, view available parking slots, and reserve a parking spot before entering the parking area.

## 📌 Problem Statement

Managing vehicles in indoor parking lots manually can lead to:
- Time-consuming parking allocation
- Human errors in slot management
- Difficulty in tracking occupied and vacant spaces
- Traffic congestion inside parking lots

This project provides a digital solution to automate the parking process.

---

## ✨ Features

- 📱 QR Code based parking access
- 🅿️ View all available parking slots
- ✅ Book a preferred vacant parking slot
- 🚘 Automatically mark booked slots as occupied
- ❌ Notify users when no parking slots are available
- 📊 Digital parking slot management
- ⚡ Simple and user-friendly interface

---

## 🛠️ How It Works

1. Driver arrives at the parking entrance.
2. Driver scans the QR code displayed at the entrance.
3. The parking application opens.
4. Available parking slots are displayed.
5. Driver selects a preferred vacant slot.
6. The selected slot is reserved and marked as occupied.
7. If all slots are occupied, the application informs the user that no parking spaces are available.
